## Exam 

* MCC (final grade computation) : 
```
let ccgrade = average(Lab3, Lab4, Lab5, homework)
in (finalnote + ccgrade)/2
```
* The final exam will be done "physically" in January. It will last three (3) hours. No authorized documents.
